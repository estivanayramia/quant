# AI Boundary

AI researches, codes, critiques, summarizes, monitors, and proposes improvements.

AI does not execute orders, size positions, exit trades, reconcile state, approve risk, change limits, override the kill switch, release quarantined strategies, or hold broker authority.

Milestone 1 uses a deterministic mock AI provider only. Future AI-provider routing may use LiteLLM, OpenRouter, NVIDIA NIM, Gemini, Groq, OpenAI, or local OpenAI-compatible endpoints for support tasks only.
